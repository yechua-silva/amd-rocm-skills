# Publishing Checklist

## Pre-publicación
- [ ] Skills completas y probadas
- [ ] README.md actualizado
- [ ] LICENSE (Apache 2.0)
- [ ] .gitignore

## Publicación en skills.sh
1. Crear repo público en GitHub
2. skills.sh detecta automáticamente repos con skills/ directory
3. Verificar: `npx skills add <owner>/<repo> --list`
4. Opcional: solicitar listing en skills.sh

## Post-publicación
- [ ] Anunciar en AMD Community
- [ ] Anunciar en r/ROCm
- [ ] Anunciar en r/AMD
- [ ] Compartir en LinkedIn
